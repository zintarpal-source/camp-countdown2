# Табір Зелем’янка — повноцінна React/Vite версія

## Запуск локально

```bash
npm install
npm run dev
```

Після запуску відкрий адресу, яку покаже Vite, зазвичай:

```text
http://localhost:5173
```

## Збірка для публікації

```bash
npm run build
```

Готові файли будуть у папці `dist`.

## Що реалізовано

- головна сторінка без верхнього меню;
- адреса: `с. Гребенів`;
- 3 клікабельні картки заїздів;
- сторінка кожного заїзду;
- живий відлік для кожного заїзду;
- блок команд після прокрутки;
- медальйони команд у пригодницькому стилі;
- адаптація під мобільні екрани.
